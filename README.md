# HomeNav – Smart Housing Finder Platform

## Project Overview
HomeNav is a full-stack web application that helps students, employees, families, and tourists discover long-term rentals, properties for sale, and short-term stays.

## Responsibilities / What I Implemented
- Designed and built a full-stack housing discovery platform
- Implemented search and filtering for rentals, sales, and short stays
- Integrated authentication, listings management, etc.

## Tech Stack
- **Frontend:** React, Next.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Other:** REST APIs, JWT, etc.

## Repository Structure
- `frontend/` – React/Next.js client application  
- `backend/` – Node.js/Express API  
- `docs/` – Documentation (if any)  

## Setup & Usage
1. Clone the repository.
2. Install dependencies in `frontend/` and `backend/`.
3. Configure environment variables.
4. Run backend and frontend servers.
